# UrbanThread

UrbanThread is a mini full-stack web project with user login and a product dashboard.

---
## Technologies Used
- HTML, CSS, JavaScript  
- Node.js & Express  
- MySQL  
- Git & GitHub  
---

## Project Structure
urbanthread-project/
├── frontend/
├── backend/
├── .gitignore
└── README.md

---
## Features
- User login and registration
- Backend authentication
- Product dashboard
- Simple and responsive UI
---

## How to Run
### Backend
cd backend
node backend.js

Backend runs on `http://localhost:3000`

### Frontend
Open `login.html` in a browser (or use Live Server).

---

## Notes

- `node_modules` is ignored using `.gitignore`
- Images are stored directly in the frontend folder
